# AI Maturity Assessment — Professional Edition

A clean, horizontally-navigated assessment with professional design quality.

## Design Principles

### 1. **Typography**
- **Sora** for UI (geometric, clean, professional)
- **Lora** for headings (elegant serif, readable)
- Scale: 48px → 36px → 18px → 16px → 15px → 13px
- Proper line height: 1.2 for headings, 1.5-1.7 for body
- Consistent letter spacing

### 2. **Spacing**
- Based on 8px grid
- Consistent gaps: 8, 12, 16, 20, 24, 32, 40px
- Generous whitespace
- Proper padding and margins throughout

### 3. **Color**
- Restrained palette (HSL-based for consistency)
- Background: hsl(220, 16%, 8%)
- Surface: hsl(220, 14%, 12%)
- Accent: hsl(217, 91%, 60%)
- Text hierarchy: 96%, 70%, 50% lightness
- Professional, not oversaturated

### 4. **Layout**
- Max-width: 640px (optimal reading width)
- Centered content
- Balanced proportions
- Clear hierarchy

### 5. **Interactions**
- 300ms transitions (feels responsive)
- Subtle transforms (2px max)
- Smooth easing (cubic-bezier)
- No jarring movements

### 6. **Details**
- 1.5px borders (subtle but visible)
- 16px border radius (modern but not extreme)
- Proper shadows (layered depth)
- Consistent hover states

## Navigation

- **Forward:** Click answer or right arrow key
- **Back:** Back button or left arrow key
- **Restart:** Top right button
- Auto-advances after selection (300ms)

## Structure

- Welcome screen → 5 questions → Results
- One screen at a time
- Slide transitions (40px offset)
- Progress bar at top

## Files

- `index.html` — Clean semantic markup
- `styles.css` — Professional styling, mobile-first
- `app.js` — Simple navigation logic

## Responsive

- Desktop: 640px centered container
- Mobile: Full width with padding
- Touch-friendly: 18px padding on buttons
- Readable: minimum 14px font size

## Quality Standards

✓ Proper typographic scale  
✓ Consistent spacing rhythm  
✓ Professional color palette  
✓ Clean visual hierarchy  
✓ Subtle, purposeful interactions  
✓ Accessibility (keyboard nav, focus states)  
✓ Mobile responsive  
✓ Fast performance
